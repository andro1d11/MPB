Вітаю у програмі MusicPyBox!

Керування треками:
    Плейлист Local містить всі треки з директорії /files
    Треки, завантажені за допомогою програми, зберігаються у директорії /files
    Подвійно натисніть на трек щоб увімкнути його.
    Shift + Виділення = виділення багатьох треків.
    Ctrl + Виділення = виділення включних треків.
    Виділіть треки та подвійно натисніть на плейлист щоб добавити їх у нього.
    Виділіть треки та перетягніть їх для зміни порядку ( не працює в плейлисті Local )

Створення плейлисту:
    Натисніть на "Новий плейлист", введіть назву плейлисту, знову натисніть на "Новий плейлист" для створення

Пошук та завантаження треків:
    Введіть назву треку та натисніть "Пошук" для пошуку треків. Дані беруться з пошуку YouTube. Виділіть 1 знайденний трек зі списку та натисніть
    "Скачати" для завантаження файлу з треком, або натисніть "Зберегти для онлайн" для завантаження посилання на трек. У другому випадку трек
    буде доступний лише онлайн

    Натисніть "Вибрати файл з треками" для завантаження треків з файлу. Файл має містити назви треків, кожен з яких повинен бути написанний
    з нового рядку у такому форматі:
        Track1
        Track 2
        T -rack 3
    Буде завантаженно перший знайдений трек до кожного треку зі списку.
    Введіть YouTube посилання на плейлист та натисніть "Скачати плейлист" для завантаження кожного треку з плейлисту.
    Введіть YouTube посилання на трек та натисніть "Скачати за посиланням" для завантаження треку за YouTube посиланням.

Оцінювання треків:
    Натисніть "Оцінити", увімкнувши потрібний трек для для оцінювання, щоб власноруч оцінити трек. 
    Всі оцінені треки будуть знаходитися у вкладці "Рейтинг" в порядку спадання
